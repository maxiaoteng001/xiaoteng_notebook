# Xiaoteng Notebook

1. 为什么vuepress而不用gitbook
    gitbook暂停了命令行工具的更新,要依赖gitbook官网工具, 之后托管和更新会有隐患

    主题更新支持不好

    vuepress可以添加登录

2. 还不能很好的配置自动化目录, 设置为固定目录, 通过`python3 vuepress_sidebar.py`来刷新配置
